Follow these steps to run web app
STEP 1: open terminal in this respository
STEP 2: start frontend server
  => run these commans
      cd todo-frontend
      npm install
      npm run start
STEP 3: start backend server
  => run these commans
      cd todo-backend
      npm install
      npm run start
STEP 4: open browser, and type url: http://localhost:3000
